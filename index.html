Welcome to Varda

This is how you manage your development environments with varda.

$ cd your_repo_root/repo_name
$ varda new papaya_node https://github.com/brunotg/papaya_node
$ git add .
$ git commit -m “Adding Vardafile”

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

Create a new development environment

It's easy to query for available images with standard configurations to get started a new development environment. Once you have chosen the platform of your preference and development framework; varda will create the customized image with the code from your git depot.

Add a new member to your team
